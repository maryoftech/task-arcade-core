# TaskArcade Core // Cognitive State Machine

Development framework powering the core task optimization mechanics of **TaskArcade**, an integrated application module within the proprietary **SOLVM™ Ecosystem Core** by **Mary of Tech Pty Ltd** (Melbourne, Australia).

## Architectural Overview

Traditional productivity applications rely heavily on continuous push notifications, persistent cloud synchronization, and invasive telemetry tracking. This architecture frequently introduces cognitive friction and operational anxiety, particularly for neurodivergent users navigating ADHD or executive dysfunction.

`task-arcade-core` isolates the underlying mathematical state logic from our proprietary user interface shells. It enforces a deterministic, local-first state model designed to regulate and support focus sprints through deliberate dopamine-feedback milestones without utilizing manipulative external data-extractive loops.

### Key Pillars
* **State Isolation:** Decoupled business logic that tracks task parameters locally on the client layer.
* **Deterministic Feedback:** Real-time event hooks triggered upon focus sprint completion to anchor cognitive momentum.
* **Zero-Telemetry Blueprint:** Structured entirely to bypass public cloud dependencies, maintaining user data privacy and sovereignty.

## Core State Lifecycle

The underlying machine processes task execution loops across four distinct lifecycle phases:

1. `IDLE`: Dormant standby state waiting for user initialization.
2. `FOCUS_SPRINT`: A highly insulated local countdown frame (default: 25-minute intervals).
3. `CELEBRATION_TRIGGERED`: A deterministic milestone hook signaling system victory parameters.
4. `COGNITIVE_REST`: An automated recovery phase protecting cognitive stamina before cycle loops reinitialize.
